# Privacy Information for ExGo

## Overview

ExGo is a privacy-first budgeting application. All your financial data is stored locally on your device. We do not collect, transmit, or store any of your financial information.

## Data Storage

### Local Storage Only

- ✅ All transaction data stored on your device
- ✅ All settings stored on your device
- ✅ No cloud sync
- ✅ No server storage
- ✅ No data transmission

### What's Stored Locally

- Transaction history (expenses, income, savings)
- Monthly income and currency settings
- Custom categories
- Theme preferences
- Security settings (biometric/PIN preferences)

## Error Tracking

We use Sentry for error tracking to improve app stability. Sentry collects:

- Error messages and stack traces
- Device information (model, OS version)
- App version
- Technical diagnostic data

**Important**: 
- ❌ No financial data is sent
- ❌ No transaction details are sent
- ❌ No personal information is sent
- ❌ Data is not linked to your identity
- ❌ Data is not used for tracking or advertising

## Security Features

### Biometric Authentication

- Face ID (iOS)
- Touch ID (iOS)
- Fingerprint (Android)
- PIN Code

All authentication is handled locally on your device. We do not store or transmit authentication credentials.

## Your Control

- **Delete Data**: Use "Reset all data" in Settings
- **Export Data**: Use PDF export feature
- **Uninstall**: Remove app to delete all data
- **Disable Error Tracking**: Not available (required for app stability)

## Third-Party Services

### Sentry

- **Purpose**: Error tracking and diagnostics
- **Data Shared**: Error logs, device info, app version
- **Privacy Policy**: https://sentry.io/privacy/
- **Data Retention**: According to Sentry's policy (typically 90 days)
