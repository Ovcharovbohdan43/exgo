# Идеи улучшений ExGo

Краткий список возможностей, которые добавят ценность пользователю и улучшат надежность приложения.

## UX и навигация
- Горизонтальная карусель месяцев с прыжком к текущему и быстрыми кнопками «Сегодня»/«Текущий месяц», чтобы упростить навигацию при длинной истории.
- Улучшенный поиск/фильтр транзакций (по категории, сумме, типу) прямо из Home/Details, чтобы быстрее находить операции.
- Пустые состояния с CTA: если нет транзакций или категории, показывать подсказки «Добавить расход», «Настроить доход».

## Финансовые фичи
- Повторяющиеся операции (аренда, подписки) с авто-добавлением по расписанию и пометкой «ожидается» до подтверждения.
- План/лимит по категориям (мягкие лимиты с цветовым индикатором и предупреждением при превышении).
- Перенос остатка: опция переносить положительный остаток предыдущего месяца в доход следующего, для более реалистичного учета.
- Цели накоплений: тип `saved` с целевой суммой и прогрессом, отображаемым в Donut/Details.

## Аналитика и отчетность
- Тренды по месяцам: мини-график расходов/сбережений за 6–12 месяцев на экране Details или отдельной вкладке.
- Экспорт/импорт CSV/JSON локально (Share API) для резервных копий и миграции на новый телефон.
- Сводка за период: быстрые пресеты (текущий месяц, прошлый, квартал) с агрегированными цифрами и топ-5 категорий.

## Настройки и безопасность
- Пин/биометрия для входа в приложение, чтобы скрыть финданные на устройстве.
- Регулировка валюты с пересчетом исторических данных по фиксированному курсу пользователя (ручной ввод) для тех, кто ведет учет в другой валюте.
- Настраиваемые категории: добавление своих категорий и эмодзи/иконок, архивирование лишних.

## Производительность и качество
- Фоновые вычисления totals/selectors с `useMemo`/`useCallback` плюс предзагрузка соседних месяцев для мгновенного свайпа.
- Диагностика хранения: экран «Состояние данных» (кол-во записей, размер, последняя синхронизация) и кнопка «Починить/очистить кэш».
- Автотесты для свайпов и границ месяцев (e2e Detox/Expo E2E) чтобы предотвратить регрессии навигации по месяцам.
